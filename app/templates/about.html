    About VulnScanner

    Learn more about this security assessment and reconnaissance toolkit.

    Overview

    VulnScanner is a web-based security assessment tool designed to assist security engineers, penetration testers, and system administrators in identifying potential security issues in web applications and network infrastructure.

    It combines multiple modules—reconnaissance, port scanning, subdomain enumeration, web crawling, vulnerability detection, and reporting—into a single, cohesive interface.

    This tool is intended to be easy to use while still exposing powerful capabilities usually found in command-line focused security toolchains.

    Key Features
    Reconnaissance
    • Port scanning with service detection
    • DNS & WHOIS intelligence
    • Subdomain enumeration
    • Technology and stack detection
    Vulnerability Scanning
    • XSS detection (reflected & DOM-oriented heuristics)
    • SQL injection detection (error, boolean, time, union)
    • Directory and file bruteforcing
    • Security headers and SSL/TLS checks
    Crawling & Forms
    • Automatic crawling with depth and scope control
    • Form discovery (login, search, upload)
    • Email and comment extraction for reconnaissance
    Reporting
    • JSON reports and history view
    • Aggregated risk scoring
    • Per-vulnerability details and remediation hints
